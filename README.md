# 🚀 EduGamify - Backend API

API RESTful para o sistema de gamificação educacional EduGamify.

## 📋 Sobre

Backend desenvolvido em Node.js/Express que gerencia:
- Alunos e pontuações
- Catálogo de recompensas
- Histórico de atividades
- Solicitações de resgate

## 🛠️ Tecnologias

- **Node.js** - Runtime JavaScript
- **Express** - Framework web
- **Supabase** - Banco de dados PostgreSQL na nuvem
- **CORS** - Compartilhamento entre domínios
- **Dotenv** - Gerenciamento de variáveis ambiente
